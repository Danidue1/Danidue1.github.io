Oh my Llama!

A webmap displaying the llama and alpaca populations for each US state. The user is given interatction with the individual states to see the Alpaca populations while the icon within each state represents the llama populus.

The interactive map is written in JavaScript utlizing the Leaflet API.

Lighting up the night sky

A webmap displaying graduated circles representing the energy production of select power plants in the U.S. The 2 colors represent natural gas produced energy in conjunction with coal and petroleum.

The interactive map is written in JavaScript utlizing the Leaflet API.

Town of Halfmoon Community Map

Created as a discovery tool for residents and the public alike this interactive web viewer allows for the search and query of various publicly available data sets. Click here to have a look for yourself!

The interactive map is written in JavaScript while utilizing mapbox and leaflet APIs. This map relies on Boostrap CSS to provide flexible interaction

The Uninsured

Interactive map displaying the US counties and their rates of uninsured populations

The interactive map harnesses the power of the CARTO platform and is written in Carto.js .
